Imagini în HTML

HTML folosește tagul <img> pentru a insera imagini. Acest tag este vid (nu are etichetă de închidere) și se folosesc doar atribute.

1. Imagine de pe internet

Exemplu cu o imagine de pe web:

Explicație:

  • src – adresa imaginii (link complet sau nume fișier)
  • alt – text alternativ afișat dacă imaginea nu se încarcă (important pentru accesibilitate)
  • width / height – controlează dimensiunile (în pixeli)

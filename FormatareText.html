Formatarea Textului în HTML

HTML oferă mai multe taguri pentru a formata textul. Iată cele mai importante:

1. Îngroșare, Italic și Subliniere

Text îngroșat cu <b>
Text îngroșat semantic cu <strong>
Text înclinat cu <i>
Text evidențiat semantic cu <em>
Text subliniat cu <u>

2. Dimensiuni relative ale textului

Text H1 (mare)

Text H2

Text H3

Text H4

Text H5
Text H6 (mic)

Text mic cu <small>

3. Centrarea textului

Acest text este centrat cu tagul <center> (vechi, dar încă funcțional).

4. Culoarea textului (cu <font> – notă: învechit)

Text roșu
Text albastru
Text verde (cod hex)

Notă: tagul <font> este învechit. Se recomandă folosirea CSS pentru culoare și fonturi în pagini moderne.

5. Alte elemente utile

Text evidențiat cu <mark>
Text tăiat cu <del>
Text adăugat cu <ins>
Exponent cu <sup> (ex: x2)
Indice cu <sub> (ex: H2O)

6. Aliniere și spațiu

HTML ignoră spațiile multiple și rândurile goale.

Pentru spațiu fix: folosim &nbsp; (spațiu neîntrerupt).

Exemplu:   Acest text are spații la început.

7. Comentarii în HTML

Comentariile nu apar în pagină, dar sunt utile în cod:

<!-- Acesta este un comentariu HTML -->

Concluzie

HTML oferă unelte simple pentru formatarea textului. Pentru control complet asupra aspectului, se folosește CSS – dar aceste taguri sunt utile pentru structurarea rapidă a conținutului.

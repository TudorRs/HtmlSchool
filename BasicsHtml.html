Noțiuni de Bază HTML

HTML (HyperText Markup Language) este limbajul principal folosit pentru a crea pagini web. Este format din taguri care structurează și definesc conținutul.

Structura de bază a unei pagini HTML

Orice pagină HTML începe cu această structură:

<!DOCTYPE html>
<html>
  <head>
    <title>Titlul paginii</title>
  </head>
  <body>
    <h1>Salut!</h1>
    <p>Acesta este un paragraf.</p>
  </body>
</html>

Taguri importante

  • <h1> - <h6>: titluri de la cel mai mare (h1) la cel mai mic (h6)
  • <p>: paragrafe
  • <a>: linkuri
  • <img>: imagini
  • <ul>, <ol>, <li>: liste
  • <div>: container generic pentru conținut

Exemplu simplu

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Pagina Mea</title>
  </head>
  <body>
    <h1>Bine ai venit!</h1>
    <p>Aceasta este prima ta pagină HTML.</p>
    <a href="https://www.google.com">Mergi la Google</a>
  </body>
</html>

Ce urmează?

După ce înveți aceste elemente de bază, poți trece la învățarea despre:

  • Formatarea textului
  • Crearea formularelor
  • Inserarea imaginilor și a videoclipurilor

Formulare în HTML

Un formular HTML permite colectarea de informații de la utilizatori. Se definește cu tagul <form>.

Explicații scurte

  • <form> – începe formularul
  • action – unde se trimit datele (ex: trimite.php)
  • method – metoda de trimitere: get sau post
  • input type="text" – câmp pentru text
  • input type="password" – câmp ascuns pentru parolă
  • radio – selectare unică dintre opțiuni
  • checkbox – bifă opțională
  • select + option – listă derulantă
  • textarea – câmp mare pentru text liber
  • submit – buton care trimite formularul
  • reset – resetează toate câmpurile

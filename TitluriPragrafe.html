Titluri și Paragrafe în HTML

În HTML, titlurile și paragrafele sunt esențiale pentru structurarea textului și pentru o bună organizare a conținutului unei pagini web.

Titlurile HTML

HTML oferă șase niveluri de titluri, de la <h1> (cel mai important) până la <h6> (cel mai puțin important).

Exemplu de utilizare a titlurilor:

<h1>Titlu Principal</h1>
<h2>Subtitlu</h2>
<h3>Sub-subtitlu</h3>
<h4>Nivel 4</h4>
<h5>Nivel 5</h5>
<h6>Nivel 6</h6>

Recomandare: folosește un singur <h1> pe pagină, care să reprezinte subiectul principal. Celelalte titluri se folosesc ierarhic pentru a structura informația.

Paragrafe în HTML

Paragrafele se definesc cu ajutorul tagului <p>. Ele sunt folosite pentru a scrie bucăți de text clar separate.

Exemplu de paragraf:

<p>Acesta este un paragraf în HTML.</p>

Fiecare paragraf este afișat pe o linie nouă. Nu este nevoie să adaugi spații sau linii goale între paragrafe – browserul le afișează automat cu spațiu între ele.

Exemplu complet

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Exemplu Titluri și Paragrafe</title>
  </head>
  <body>
    <h1>Bine ai venit!</h1>
    <p>Aceasta este prima mea pagină HTML.</p>
    <h2>Despre mine</h2>
    <p>Îmi place să învăț HTML și să creez pagini web.</p>
  </body>
</html>

Concluzie

Tagurile <h1>–<h6> și <p> sunt fundamentale pentru orice pagină HTML. Ele oferă structură, claritate și ajută la o bună organizare a conținutului.

Ce este <meta charset="UTF-8">?

Linia <meta charset="UTF-8"> se pune în secțiunea <head> a unei pagini HTML.

Ce face?

Spune browserului că pagina folosește codificarea de caractere UTF-8, care suportă literele românești (ă, î, â, ș, ț) și multe altele.

De ce este important?

  • Fără această linie, caracterele speciale pot apărea greșit.
  • Asigură afișarea corectă a textului în orice browser.
  • Este recomandat pentru orice pagină HTML modernă.

Exemplu de structură corectă

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Exemplu cu UTF-8</title>
  </head>
  <body>
    <p>Salut! Aceasta este o pagină cu diacritice: ă, î, â, ș, ț.</p>
  </body>
</html>

Concluzie

Folosirea <meta charset="UTF-8"> este simplă și esențială pentru ca textul să se vadă corect în orice limbă.
